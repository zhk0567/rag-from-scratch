# 项目说明

**rag-from-scratch** 是一个从零搭建的检索增强生成（RAG）演示项目：把本地文档变成可检索的知识库，再根据用户问题检索相关片段，交给大模型生成有据可依的回答。

## 一句话

上传文档 → 建向量索引 → 提问 → 基于检索结果由 Ollama 本地回答，并在界面展示引用来源。

## 特点

- **本地运行**：嵌入与对话均通过 Ollama，无需云端 API Key
- **轻量实现**：不依赖 LangChain；向量检索用 NumPy 自写余弦相似度
- **可视化演示**：Streamlit 网页支持上传文档、重建索引、聊天与查看引用片段

## 技术栈

Python · Ollama · NumPy · Streamlit · pypdf

## 适用场景

教学演示、小体量私有文档问答、理解 RAG 基本流程（加载 → 分块 → 嵌入 → 检索 → 生成）。
