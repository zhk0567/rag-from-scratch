# 可选依赖

| 文件 | 用途 | 安装 |
|------|------|------|
| `base.txt` | 核心依赖（根目录 `requirements.txt` 已引用） | `pip install -r requirements.txt` |
| `ocr.txt` | 扫描版 PDF OCR | `pip install -r requirements/ocr.txt` |
| `clip.txt` | CLIP 图文双索引 | `pip install -r requirements/clip.txt` |
| `video.txt` | 视频抽帧 | `pip install -r requirements/video.txt` |
| `qdrant.txt` | Qdrant 向量库 | `pip install -r requirements/qdrant.txt` |
| `milvus.txt` | Milvus 向量库 | `pip install -r requirements/milvus.txt` |
